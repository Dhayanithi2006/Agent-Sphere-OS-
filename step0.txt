Final Demo (Hackathon Flow)

Your 3-minute demo should tell a compelling story:

User enters a prompt: "Create a 2-minute animated product advertisement."
The Microkernel boots a new process and assigns it a PID.
The Supervisor decomposes the work into specialized agents (Writer → Storyboard → Scene → Video → Voice → Editor → Reviewer).
The Scheduler runs independent tasks in parallel where possible.
The Model Router intelligently chooses the right Qwen model (e.g., Qwen-Max for planning, Qwen-Coder for code, Wan for video generation).
The Checkpoint Manager saves progress after every stage.
Simulate a failure in the Video Agent to demonstrate the Recovery Engine, which restores only that stage instead of restarting the entire pipeline.
The Dashboard visualizes PIDs, dependency graphs, events, token usage, and API costs in real time.
The finished video is stored in Alibaba OSS, and the user receives the final output.
